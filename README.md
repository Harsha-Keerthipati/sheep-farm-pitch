# Sheep Farm — Investor Pitch

A private, single-page investor pitch for a commercial sheep fattening business in
KR Kandriga, Annamayya District, Andhra Pradesh.

- **Single file:** everything lives in [`index.html`](index.html) — no build step, no dependencies to install.
- **Stack:** plain HTML + Tailwind CSS (CDN) + a little vanilla JS for scroll reveals.
- **Hosting:** GitHub Pages.
- **Privacy:** `robots.txt` and a `noindex` meta tag keep it out of search engines. Share the link only with the intended reader.

## Editing the placeholders

Open `index.html` and search for `PLACEHOLDER` (HTML comments mark each one):

| Placeholder | What to put |
|-------------|-------------|
| `[OWNER_NAME]` | Owner's name (appears on the CTA button and footer) |
| `[FRIEND_NAME]` | The friend/partner this is shared with (footer note) |
| `[WHATSAPP_NUMBER]` | `wa.me` link — country code + number, e.g. `919876543210` |
| `[PHONE_DISPLAY]` | Phone number shown as plain text |
| `[SITE_URL]` | Final GitHub Pages URL (optional) |

After editing, commit and push — GitHub Pages redeploys automatically.
